### コメント

関数を使うことでコードを見やすくしたり、同じ操作をするコードを何回も書かなくて済みます。
また、後に学習する**再帰関数**という関数を使った重要なテクニックもありますので、しっかり関数について理解しましょう。
関数の書き方をここに記しておきます。
```
(返り値の型) (関数の名前)((引数)){
    (処理)
}
```
